Odoo Google Contact Connector

Odoo is an Enterprise Resource Planning(ERP) system which includes CRM, Inventory, Warehouse etc. Google is well known company who develeops Mobile Operating System, famous Search Engine and internet related services. Odoo integration with Google Apps make it easy to sync the data i.e Contacts both ways.

  • Installation
  • How its work

Installation

  • Download ODOO Google Contact Connector module and place into addons directory
  • Restart Odoo server / service
  • Open setting, activate Developer Mode
  • Go to Apps
  • Update Apps List from top menu
  • Search the ODOO Google Contact Connector and install it
  • Happy to use Google Cloud Apps


How Its Work


  • Login to Google Account
  • Go to Google Developer Console
  • Create new project

  • Fill Project name field
  • Click on Create button

  • Config OAuth Consent
  • Go to OAuth Consent Screen
  • Select External option
  • Click on Click button

  • Fill Application name field
  • Select Support email from options

  • Fill Developer Support email addresses
  • Click on Save & Continue button

  • Optional Scope options

  • Click on Save & Continue button

  • Optional add Test users
  • Click on Save & Continue button

  • Review all project information
  • Click on Back to Dashboard button

  • Click on Publish App button

  • Create credentials account
  • Click on Credentials

  • Click on Create Credentials button

  • Select Application type (Web Application)
  • Fill Application Name

  • Add Redirect URI, which will redirect back after authorization
  • Fill in the Redirect URI like [ODOO-URL]/google_success
  • Click on Create button

  • Collect information from credentials (ClientID, ClientSecret, RedirectURI)

  • Enable the following Service APIs permission in Google Developer Console
  • (Contact API, People API, Gmail API, Google Drive API, Task API, Calendar API)

  • Install ODOO Google Connector Module
  • Go to Google Connect from menu
  • Fill credential values collected from Google Developer Console
  • Click on Connect button
  • It will open an authorization page
  • After authentication, it will redirect to Redirect-URI(ODOO page).

  • Import History menu shows how many records are imported
  • There will be no record if nothing is imported.

  • Export History menu show how many records are exported
  • There will be no record if nothing is exported.
